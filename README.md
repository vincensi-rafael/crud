# 💾 CRUD em Java (Console)

Um sistema de gerenciamento de usuários em Java baseado em operações CRUD (Create, Read, Update, Delete), com persistência de dados em arquivo local.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar os conceitos fundamentais de programação em Java, incluindo:

* Estruturas de controle
* Manipulação de arquivos
* Organização de código
* Lógica de CRUD

O sistema funciona via terminal, permitindo o gerenciamento de usuários de forma simples e direta.

---

## ⚙️ Funcionalidades

* ✅ Criar usuário
* 📋 Listar usuários
* ✏️ Atualizar usuário
* ❌ Remover usuário
* 💾 Persistência em arquivo (`usuarios.txt`)

---

## 💡 Exemplo de uso

```bash
1 - Criar usuário
2 - Listar usuários
3 - Atualizar usuário
4 - Remover usuário
Escolha uma opção:
```

---

## 🛠️ Tecnologias utilizadas

* Java
* Programação orientada a objetos
* Manipulação de arquivos (File / FileWriter / BufferedReader)

---

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/vincensi-rafael/crud.git
```

2. Acesse a pasta:

```bash
cd crud
```

3. Compile o projeto:

```bash
javac src/Main.java
```

4. Execute:

```bash
java -cp src Main
```
